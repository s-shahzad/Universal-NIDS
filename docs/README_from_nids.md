# NIDS

Network Intrusion Detection System (NIDS) repository organized for clean collaboration and GitHub readability.

## Project Structure

```text
NIDS/
  reports/
  assets/
    screenshots/
    images/
  src/
  notebooks/
  data/
    raw/
    processed/
  archives/
  docs/
  scripts/
  config/
  README.md
  .gitignore
```

## Notes

- Python source files are under `src/`.
- Datasets are under `data/raw/`; generated outputs should go to `data/processed/`.
- Reports and visuals are separated into `reports/` and `assets/`.
- Move history is tracked in `docs/file_moves.csv`.
